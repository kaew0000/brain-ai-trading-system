# World HQ — Asset Attribution

All graphical assets in this module are **procedurally generated** at runtime
using the HTML5 Canvas API. No external image files are loaded or distributed.

## Color Palette Inspiration

The cyberpunk color palette (deep navy backgrounds, neon green/cyan accents,
amber warnings) is inspired by the **Kenney 1-Bit Pack** color system.

- **Author:** Kenney Vleugels (kenney.nl)
- **License:** CC0 1.0 Universal (Public Domain)
- **URL:** https://kenney.nl

## Fonts

All in-game text uses **system monospace** fonts (Courier, Monaco, Consolas)
which are universally available — no external font files required.

## Audio (optional)

If audio is enabled, the system generates tones using the
**Web Audio API** oscillator nodes — no audio files loaded.
